Convert your audio files to different formats

Learn how to convert your audio files to smaller formats using a bash script

This script was created to convert 24bit FLAC files to 16bit FLAC, MP3 320, and MP3 V0 files. If you only have 16bit FLAC files, it'll skip the first step and convert them into MP3 320 and MP3 V0 instead.

Unsure how to check the quality of your audio files? You can review them using spek!

Dependencies

  • ffmpeg - Installed on most Linux distributions.
  • LAME - Installed on most Linux distributions.
  • FLAC - Installed on most Linux distributions.
  • SoX - Installed on most Linux distributions.
  • Perl - Installed on most Linux distributions.
  • spsox - Follow the installation instructions on GitHub.
  • torf - Follow the installation instructions on GitHub.
  • torf-cli - Follow the installation instructions on GitHub.

NOTE: This script was made with Linux in mind, but macOS users may have success. While both SoX and spsox are Linux exclusive, they have versions available on Homebrew.

Installation

If you'd like to preview the script before downloading the zip file, you can do so here.

  1. Download the zip file.
  2. In the root directory, make flac2mp3.pl executable. 
    $ chmod +x flac2mp3.pl
  3. In the FLAC directory, make work.sh executable. 
    $ chmod +x work.sh

This script uses FLAC2MP3 to convert the audio files, which comes packaged in the zip file.

Usage

Your files must be in FLAC before starting. If they're still in WAV, you can run the following command in the directory with your WAV files. 

$ mkdir -p wav_files flac_files && for file in *.wav; do ffmpeg -i "$file" "flac_files/${file%.wav}.flac" && mv "$file" wav_files/; done

  1. Move your .flac files and cover image into the FLAC directory.
  2. Open terminal in the FLAC directory and run work.sh.
  3. Fill out the details as asked by the prompts.
  4. Wait for the script to finish, then check the Complete directory.
